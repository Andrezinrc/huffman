# huffman

Projeto simples em C para estudar compressão e descompressão usando o algoritmo de Huffman.

O código implementa:

- Contagem de frequência
- Construção da árvore de Huffman
- Geração da tabela de códigos
- Compressão (encode)
- Descompressão (decode)

O objetivo é apenas aprendizado sobre estruturas, árvores e manipulação de bits em C.

## Compilar
```bash
gcc main.c huffman.c -o huffman
```
## Executar
```bash
./huffman
```

O programa mostra o tamanho original, o tamanho comprimido, os bytes gerados e reconstrói o texto original.
